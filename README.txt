~~ EZResign by MotoLegacy ~~

Prerequisites:
	1. Python 3.X, 2.7
	2. WINE (For Linux users, UNTESTED!)
	3. A dumped IDPS.hex and ACT.dat (place in bin folder)

Usage:
	1. Place all .PKGs and .RAPs in the same directory as EZResign.py
	2. Run EZResign.py with Python 3
	3. After the script has completed, put ALL files in the output folder on a USB drive
	4. Install the .PKGs with your PS3's Package Manager
